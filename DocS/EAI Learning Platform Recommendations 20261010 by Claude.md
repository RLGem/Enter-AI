### מה צריך:
1. **YouTube** - סרטונים (Unlisted = לא רשום)

### איך עובד:

```
┌─────────────────────────────────────────────────────────────────────────┐
│ 📹 תלמיד מקבל:                                                         │
│                                                                         │

│ 2. לינק ל-Google Doc (חוברת עבודה)                                     │
│    → "יצירת עותק" → ממלא תוך כדי צפייה                                 │
│ 3. (אופציונלי) טופס סיום לאישור                                        │
│                                                                         │

└─────────────────────────────────────────────────────────────────────────┘
```

### יתרונות:
- ✅ חינם לגמרי
- ✅ אפס התקנות
- ✅ כולם מכירים את הכלים
- ✅ קל לעדכן
- ✅ אפשר לראות מה התלמידים כותבים (אם משתפים)
- ✅ עובד על כל מכשיר

### חסרונות:

- ❌ אין תעודות/אישורים אוטומטיים

# **🟡 אופציה 2: WordPress LMS**

## LearnPress או Tutor LMS

### השוואה:

| תכונה | LearnPress | Tutor LMS |
|:-------|:-----------|:----------|
| מחיר בסיס | חינם | חינם |
| ממשק | פשוט | מודרני יותר |
| בונה קורסים | Drag & Drop | Drag & Drop |
| חידונים | כן | כן (10+ סוגים) |
| תעודות | תוסף בתשלום | Pro |
| AI מובנה | כן ✨ | לא |
| תמיכה בעברית | חלקית | חלקית |
| Pro מחיר | ~$69 (חד פעמי) | ~$199/שנה |

### המלצה: **LearnPress**

למה?
- חינמי לחלוטין לתחילת דרך
- תשלום חד-פעמי (לא מנוי שנתי)
- AI מובנה ליצירת תוכן
- קל להתקנה

### איך להתקין:

```
1. WordPress → Plugins → Add New
2. חפש "LearnPress"
3. Install → Activate
4. עקוב אחרי ה-Setup Wizard
5. צור קורס ראשון
```

### עלות: **חינם** (Pro: $69 חד-פעמי)
