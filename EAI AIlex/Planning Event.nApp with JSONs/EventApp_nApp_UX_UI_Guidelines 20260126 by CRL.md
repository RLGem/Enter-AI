## Discovery Prompts לפי הקשר

### אחרי הצגת Dashboard:
```
💡 רוצה משהו יותר מפורט?
   • "תראה גרף התקדמות לאורך השבוע"
   • "מי הכי הרבה עמד ביעדים?"
   • "מה צריך לעשות עד סוף החודש?"
```

### אחרי עדכון משימה:
```
💡 מעולה! עכשיו אפשר גם:
   • "תעדכן את שרה שסיימתי"
   • "מה המשימה הבאה שלי?"
   • "תכין סיכום של מה שעשיתי השבוע"
```

### אחרי יצירת משימה:
```
💡 רוצה לדייק עוד?
   • "תוסיף checklist פנימי למשימה"
   • "תציע לי תאריך יעד סביר"
   • "מה עוד שוכחים בדרך כלל בנושא הזה?"
```

### אחרי צפייה ברשימת משימות:
```
💡 יש עוד דרכים לראות את המידע:
   • "תסדר לפי דחיפות"
   • "תראה רק את מה שחסום"
   • "תן לי ציר זמן ויזואלי"
```

### אחרי דוח סטטוס:
```
💡 רוצה ניתוח מעמיק יותר?
   • "למה צד החתן מתעכב?"
   • "מה הסיכון הכי גדול שלנו עכשיו?"
   • "תשווה אותנו לאירועים אחרים"
```

### כשיש בעיה:
