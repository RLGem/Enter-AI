# 👋 מה זה Event.nApp?

**Event.nApp** הוא עוזר AI לתכנון אירועים משפחתיים.

אבל הוא **לא אפליקציה רגילה** - אין כפתורים, אין תפריטים.
פשוט **מדברים** איתו בשפה טבעית! 🗣️

---

# 💬 איך זה עובד?

```
במקום ללחוץ על כפתורים...    → פשוט אומרים מה רוצים!

❌ "חיפוש כפתור יצירת משימה"  → ✅ "תזכיר לאמא לסגור צלם"
❌ "איפה הדוח של צד הכלה?"    → ✅ "מה הסטטוס של צד הכלה?"
❌ "איך מייצאים לאקסל?"       → ✅ "תייצא לי את זה לאקסל"
```

# 👥 מי יכול להשתמש?

## 👑 מנהל/ת פרויקט (PM)
- בד"כ הכלה, החתן, או הורה
- יכול/ה לעשות **הכל**
- ליצור משימות, להקצות לאחרים, לראות את כל המידע

## 👥 חבר/ת צוות
- בני משפחה שמעורבים בתכנון
- יכול/ה לראות ולעדכן משימות שלו/ה
- לראות סטטוס כללי

## 👀 צופה
- משפחה רחבה, חברים
- יכול/ה לראות פרטי אירוע ולוח זמנים

עברית, אנגלית, ערבית, רוסית - מה שנוח לך.

## 4️⃣ גמיש לחלוטין

## 5️⃣ חכם
לא רק רשימות - גם ייעוץ, כתיבה, ניתוחים.

```

🔵 בתהליך:
```

## "אפשר בלי אינטרנט?"
צריך חיבור לאינטרנט כדי לדבר איתי.
