# 1️⃣ מה זה Event.nApp?

## 🎯 בקצרה

Event.nApp הוא עוזר AI לתכנון אירועים משפחתיים - חתונות, בר/בת מצווה, ימי הולדת.

## 💡 מה מיוחד?

```
┌─────────────────────────────────────────────────────────────┐
│ 🗣️ ממשק שיחה - פשוט מדברים!                              │
│    • אין כפתורים                                           │
│    • אין תפריטים                                           │
│    • פשוט כותבים מה צריך                                   │
│                                                             │
│ 👨‍👩‍👧‍👦 עבודה משפחתית                                          │
│    • כל אחד נכנס ורואה את המשימות שלו                      │
│    • עדכונים מרוכזים                                       │
│    • תיאום בין צד הכלה וצד החתן                            │
│                                                             │
│ 🤖 AI חכם                                                  │
│    • כותב מיילים לספקים                                    │
│    • נותן טיפים                                            │
│    • מזהה סיכונים                                          │
│    • מנהל תקציב                                            │
└─────────────────────────────────────────────────────────────┘
```

## 🆚 ההבדל מאפליקציות רגילות

```
❌ אפליקציות רגילות:
   לחפש כפתור → לפתוח תפריט → למלא טופס → לשמור

✅ Event.nApp:
   "תוסיף משימה לאמא לסגור צלם עד יום שישי"
   → נוצרה משימה, הוקצתה, נקבע דדליין. סיימנו!
```

# 2️⃣ איך מתחילים?

## 🔧 מה צריך

```
┌─────────────────────────────────────────────────────────────┐
│ דרישות:                                                    │
│                                                             │
│ 1. 📁 תיקייה משותפת (Google Drive / Dropbox)              │
│    → כדי שכל המשפחה תוכל לגשת לקבצים                      │
│                                                             │
│ 2. 🤖 גישה ל-Claude                                        │
│    → claude.ai או אפליקציה                                 │
│                                                             │
│ 3. 📄 קובץ הפרומפט:                                        │
│    • Event_nApp_Prompt_v4.md                               │
└─────────────────────────────────────────────────────────────┘
```

## 🚀 צעדים להתחלה

### למנהל/ת הפרויקט (פעם ראשונה):

```
1. פתחו שיחה חדשה ב-Claude
2. העלו את קובץ Event_nApp_Prompt_v4.md
3. כתבו "שלום" או "Event.nApp"
4. ענו על השאלות:
   • מה השם שלך?
   • האם את/ה מנהל/ת הפרויקט? → כן
   • איזה אירוע? (חתונה/בר מצווה/וכו')
   • מתי האירוע?
   • מי חוגג?
5. הורידו את קבצי state.json ו-events.jsonl
6. שמרו בתיקייה המשותפת
```

### לחברי צוות (אחרי שהמערכת הוגדרה):

```
1. פתחו שיחה חדשה ב-Claude
2. העלו את Event_nApp_Prompt_v4.md + state.json + events.jsonl
3. כתבו "שלום, אני [השם שלך]"
4. תראו את המשימות שלכם ותוכלו לעדכן
```

# 3️⃣ דוגמאות מלאות לפי קטגוריה

## 📋 A. ניהול משימות

### יצירת משימה פשוטה
```
👤 "תוסיף משימה לסגור צלם"

🤖 ✅ נוצרה משימה: לסגור צלם
   📅 דדליין: לא נקבע
   👤 מוקצה ל: לא הוקצה
   
   💡 למי להקצות? מתי הדדליין?
```

### יצירת משימה מלאה
```
👤 "תוסיף משימה לאמא: לבחור פרחים עד 15/2"

🤖 ✅ נוצרה משימה: לבחור פרחים
   📅 דדליין: 15 בפברואר 2026
   👤 מוקצה ל: אמא רחל
   
   💡 סימנתי לאמא. רוצה שאוסיף תזכורת?
```

### יצירת כמה משימות בבת אחת
```
👤 "תוסיף משימות: לסגור DJ, לבחור עוגה, לאשר הזמנות"

🤖 ✅ נוצרו 3 משימות:
   1. לסגור DJ | לא מוקצה
   2. לבחור עוגה | לא מוקצה
   3. לאשר הזמנות | לא מוקצה
   
   💡 למי להקצות כל אחת?
```

### עדכון סטטוס
```
👤 "סגרנו את הצלם!"

🤖 🎉 מעולה! סימנתי את "לסגור צלם" כהושלם.
   
   📈 ההתקדמות הכללית: 45% → 48%
   👰 צד הכלה: 52% → 55%
   
   💡 רוצה לעדכן פרטים על הצלם שנבחר?
```

### לראות משימות לפי אדם
```

🤖 📋 המשימות של אמא רחל:

   🔴 באיחור (1):

   🔵 בתהליך (2):
   • לבחור פרחים
   • לתאם עם אולם על סידורים
   
   ⚪ לא התחיל (3):
   • לאשר עיצוב הזמנות
   • לסגור DJ
   • לבחור עוגה
   
   ✅ הושלם (2):
   • לסגור צלם ✓
   • לבחור שמלה לאמא ✓
   
   📊 סה"כ: 8 משימות | 2 הושלמו | 1 באיחור
   
   💡 רוצה לשלוח תזכורת על התפריט?
```

### לראות משימות באיחור
```

🤖 ⚠️ משימות באיחור (3):

   🟡 איחור קל (1-3 ימים):
   • לאשר תפריט (3 ימים) - אמא רחל
   
   💡 רוצה שאשלח תזכורות? או להאריך דדליינים?
```

### לראות משימות לפי קטגוריה
```
👤 "מה המצב עם הספקים?"

🤖 📋 משימות בקטגוריית ספקים:

   ✅ הושלם (3):

   • לסגור וידאו ✓ (סטודיו אור)

```
