│                                                                            │
│ M - Money (כסף)                                                            │
│     האם אנשים משלמים היום לפתור את זה? כמה?                              │

│                                                                            │

│     האם אתה יכול לפתור את זה? יש לך יתרון?                               │

│                                                                            │
