### How to Price

```
┌────────────────────────────────────────────────────────────────────────────┐
│ 💵 איך קובעים מחיר:                                                       │
│                                                                            │
│ 📊 3 גישות:                                                               │
│                                                                            │
│ 1️⃣ COST-PLUS (עלות + מרווח)                                              │
│    • עלות לתפעל לקוח: X                                                 │
│    • מרווח רצוי: Y%                                                      │
│    • מחיר = X × (1 + Y%)                                                 │
│    ❌ לא מומלץ - מתעלם מערך ללקוח                                       │
│                                                                            │
│ 2️⃣ COMPETITION-BASED (לפי תחרות)                                         │
│    • מה המתחרים גובים?                                                   │
│    • תהיה ±20% מהם                                                       │
│    ⚠️ סביר - אבל לא מבדל                                                │
│                                                                            │
│ 3️⃣ VALUE-BASED (לפי ערך) ✅ מומלץ!                                       │
│    • כמה הלקוח חוסך/מרוויח?                                             │
│    • גבה 10-20% מהערך הזה                                               │
│    • דוגמה: חוסך 10 שעות בשבוע?                                         │
│      → 10 × 100₪ = 1000₪ ערך                                            │
│      → גבה 100-200₪                                                      │
│                                                                            │
│ ───────────────────────────────────────────────────────────────────────   │
│                                                                            │
│ 🧪 הדרך הטובה ביותר: לשאול לקוחות!                                       │
│                                                                            │
│ שאלות תמחור:                                                              │
│ • "באיזה מחיר זה יהיה יקר מדי?"                                         │
│ • "באיזה מחיר זה יהיה זול מדי (חשוד)?"                                  │
│ • "באיזה מחיר זה יהיה יקר אבל שווה?"                                    │
│ • "באיזה מחיר זה מציאה?"                                                │

│ → Van Westendorp Price Sensitivity Meter                                  │
└────────────────────────────────────────────────────────────────────────────┘
```

```
│                                                                            │
│ ───────────────────────────────────────────────────────────────────────   │
│                                                                            │
│ 🎁 סוגי Referral Programs:                                                │
```

```
│    = סה"כ הוצאות שיווק / מספר לקוחות חדשים                             │

│    🎯 יעד: LTV > 3× CAC                                                  │
│                                                                            │
│ 📈 MRR - Monthly Recurring Revenue                                        │
│    = סה"כ הכנסות חודשיות ממנויים                                       │
│    🎯 יעד: גדל כל חודש                                                  │
```
