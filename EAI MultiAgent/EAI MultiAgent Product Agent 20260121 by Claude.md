### MoSCoW Prioritization

```
┌────────────────────────────────────────────────────────────────────────────┐
│ 🎯 MoSCoW Prioritization:                                                 │
│                                                                            │
│ M - MUST HAVE (חובה) 🔴                                                   │
│     בלי זה המוצר לא עובד. אי אפשר לצאת בלי.                              │
│     דוגמה: אפשרות להוסיף משימה                                           │
│                                                                            │
│ S - SHOULD HAVE (צריך) 🟠                                                 │
│     חשוב מאוד, אבל אפשר לצאת בלי אם חייבים.                              │
│     דוגמה: תזכורות Push                                                  │
│                                                                            │
│ C - COULD HAVE (אפשר) 🟡                                                  │
```
