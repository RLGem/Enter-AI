# 1️⃣ מהות הפרויקט

## מה זה Event.nApp?

**Event.nApp** הוא עוזר AI לתכנון אירועים שפועל כממשק שיחה (צ'אט) בלבד.

```
┌────────────────────────────────────────────────────────────┐
│                                                            │
│   nApp = "No App" (עברית) + "New App" (אנגלית)            │
│                                                            │
│   הרעיון: אין אפליקציה מסורתית עם ממשק משתמש             │
│   יש שיחה טבעית עם AI שעושה הכל                          │
│                                                            │
└────────────────────────────────────────────────────────────┘
```

## מה זה לא?

- ❌ לא אפליקציה עם טפסים ודרופדאונים
- ❌ לא מערכת עם קטגוריות וסוגי אירועים מוגדרים מראש
- ❌ לא תוכנה שדורשת הגדרות לפני שימוש

## מה זה כן?

- ✅ שיחה טבעית שמתאימה את עצמה לכל אירוע
- ✅ AI שלומד מהשיחה ולא שואל שאלות מיותרות
- ✅ גמישות מלאה - כל סוג אירוע, כל סוג משימה

# 2️⃣ פילוסופיית nApp

## העקרונות המנחים

### 1. למידה מהשיחה (לא קונפיגורציה)

```
┌────────────────────────────────────────────────────────────┐
│                                                            │
│   ❌ גישה מסורתית:                                        │
│   "בחר סוג אירוע: [חתונה ▼]"                              │
│   "בחר צד: [חתן ▼] [כלה ▼]"                               │
│                                                            │
│   ✅ גישת nApp:                                           │
│   משתמש: "חתונה של רפי ורחל"                              │
│   AI למד: אירוע=חתונה, חתן=רפי, כלה=רחל                  │
│                                                            │
│   משתמש: "אני דני, אבא של רפי"                            │
│   AI למד: משתמש=דני, תפקיד="אבא של החתן", צד=חתן         │
│                                                            │
└────────────────────────────────────────────────────────────┘
```

### 2. שואלים רק כשצריך

```
┌────────────────────────────────────────────────────────────┐
│                                                            │
│   ❌ לא לשאול מראש:                                       │
│   "מה התאריך? מה התקציב? כמה אורחים?"                    │
│                                                            │
│   ✅ לשאול כשרלוונטי:                                     │
│   משתמש: "תזכיר לי לסגור צלם"                             │
│   AI: "אוסיף משימה. לאיזה תאריך לשים תזכורת?"            │
│                                                            │
└────────────────────────────────────────────────────────────┘
```

### 3. גמישות מלאה

│                                                            │

│   • "בר מצווה של יואב"                                    │
│   • "מסיבת יום הולדת 50 לאמא"                             │
│   • "הסרת הגבס מהאצבע"                                    │

│                                                            │

│   • "בחודש אוקטובר"                                       │
│   • "כשיהיה חם"                                           │
│   • "שבוע הבא ביום חמישי בשעה 21:00"                     │

│      ↓                                                     │
│   2. AI בודק: יש ספקים רלוונטיים ברשימה?                  │
│      ↓                                                     │
│   ┌─────────────┬──────────────────────────────────────┐  │
│   │ אין ❌      │ יש ✅                                 │  │
│   │             │                                      │  │

│   │ "אין לי    │  רוצה שיפנו אליך?"                   │  │
│   │  ספקים     │                                      │  │
