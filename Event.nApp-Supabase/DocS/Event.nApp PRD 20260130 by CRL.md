# 1️⃣ סקירת מוצר

## תיאור

Event.nApp הוא עוזר AI לתכנון אירועים שפועל כשיחת צ'אט בלבד.

## הבעיה שפותרים

```
┌────────────────────────────────────────────────────────────┐
│                                                            │
│   תכנון אירועים היום = כאוס:                              │
│                                                            │
│   • אקסלים מפוזרים                                        │
│   • הודעות וואטסאפ אינסופיות                              │
│   • שוכחים משימות קריטיות                                 │
│   • קשה לתאם בין כל המעורבים                              │
│   • אין תמונה אחת ברורה                                   │
│                                                            │
└────────────────────────────────────────────────────────────┘
```

## הפתרון

```
┌────────────────────────────────────────────────────────────┐
│                                                            │
│   AI אחד שיודע הכל:                                       │
│                                                            │
│   • זוכר את כל מה שנאמר                                   │
│   • מנהל משימות                                           │
│   • מתאם בין משתתפים                                      │
│   • מזכיר כשצריך                                          │
│   • מציע ספקים (כשיש)                                     │
│                                                            │
└────────────────────────────────────────────────────────────┘
```

# 2️⃣ מימדי המוצר

## המימדים החיוניים

```
┌────────────────────────────────────────────────────────────┐
│                                                            │
│   1️⃣  אירוע (Event)                                       │
│       • שם - מה שהמשתמש אומר                              │
│       • תאריך - גמיש (מדויק / חודש / "כשיהיה חם")         │
│       • תיאור - מה שלמדנו מהשיחה                          │
│                                                            │
│   2️⃣  משימות (Tasks)                                      │
│       • כותרת - חובה                                      │
│       • סטטוס - פתוח / סגור                               │
│       • כל השאר אופציונלי ובטקסט חופשי:                   │
│         - תיאור                                           │
│         - תאריך יעד                                       │
│         - אחראי                                           │
│         - קטגוריה                                         │
│                                                            │
│   3️⃣  משתתפים (Members)                                   │
│       • שם                                                │
│       • תפקיד - טקסט חופשי ("אבא של החתן", "מארגנת")      │
│       • צד/קבוצה - רק אם רלוונטי ועלה בשיחה              │
│                                                            │
│   4️⃣  ספקים (Suppliers)                                   │
│       • שם                                                │
│       • איש קשר + פרטים (מייל, טלפון)                     │
│       • אפיון - מה מספק (טקסט חופשי)                      │
│       • [רשימה מנוהלת ע"י האדמין]                         │
│                                                            │
│   5️⃣  לידים (Leads)                                       │
│       • קישור בין משתמש לספק                              │
│       • סטטוס (נשלח / נענה / סגור)                        │
│                                                            │
│   6️⃣  זיכרון (Memory)                                     │
│       • אירוע - משותף לכל המשתתפים                        │
│       • משתמש - פרטי לכל אחד                              │
│                                                            │
│   7️⃣  שיחה (Conversation)                                 │
│       • היסטוריית צ'אט לכל משתמש                          │
│       • הכל קורה בשיחה                                    │
│                                                            │
└────────────────────────────────────────────────────────────┘
```

# 4️⃣ זרימות משתמש

## זרימה 1: משתמש ראשון יוצר אירוע

```
┌────────────────────────────────────────────────────────────┐
│                                                            │
│   1. משתמש פותח צ'אט עם הבוט                              │
│      ↓                                                     │
│   2. AI מברך ומסביר מה הוא יכול לעשות                     │
│      ↓                                                     │
│   3. משתמש מספר על האירוע                                 │
│      ↓                                                     │
│   4. AI לומד ושומר (שם אירוע, סוג)                        │
│      ↓                                                     │
│   5. AI שואל את שם המשתמש                                 │
│      ↓                                                     │
│   6. משתמש מזדהה (שם + תפקיד)                             │
│      ↓                                                     │
│   7. AI שומר ושואל במה להתחיל                             │
│      ↓                                                     │
│   8. עבודה רגילה (משימות, שאלות, וכו')                    │
│                                                            │
└────────────────────────────────────────────────────────────┘
```

## זרימה 2: משתמש נוסף מצטרף

```
┌────────────────────────────────────────────────────────────┐
│                                                            │
│   1. משתמש פותח צ'אט עם הבוט                              │
│      ↓                                                     │
│   2. AI מזהה שיש כבר אירוע פעיל                           │
│      (לפי מספר טלפון / קישור / קוד)                       │
│      ↓                                                     │
│   3. AI מברך ומציג את האירוע                              │
│      ↓                                                     │
│   4. AI שואל מי המשתמש                                    │
│      ↓                                                     │
│   5. משתמש מזדהה                                          │
│      ↓                                                     │
│   6. AI שומר ומציע לראות משימות                           │
│      ↓                                                     │
│   7. עבודה רגילה                                          │
│                                                            │
└────────────────────────────────────────────────────────────┘
```

## זרימה 3: ליד לספק

```
┌────────────────────────────────────────────────────────────┐
│                                                            │
│   1. AI מזהה צורך בספק (מהשיחה)                           │
│      ↓                                                     │
│   2. AI בודק אם יש ספקים רלוונטיים                        │
│      ↓                                                     │
│   3. אם יש - מציע למשתמש                                  │
│      ↓                                                     │
│   4. משתמש מאשר                                           │
│      ↓                                                     │
│   5. AI שולח ליד לספקים (מייל/SMS)                        │
│      ↓                                                     │
│   6. AI שולח עותק לאדמין (רפי)                            │
│      ↓                                                     │
│   7. AI מאשר למשתמש                                       │
│                                                            │
└────────────────────────────────────────────────────────────┘
```
